# Collapse All Folders

An Obsidian plugin that lets you collapse or expand all folders in the file explorer with a single command.

## Features

- **Command: Collapse all folders** — Collapses every folder in the file explorer.
- **Command: Expand all folders** — Expands every folder in the file explorer.
- **Ribbon icon** — Click the folder-minus icon in the sidebar to collapse all folders instantly.
- **Auto-collapse on vault open** — Optionally collapse all folders automatically when you open your vault.

## Installation

### From Community Plugins

1. Open **Settings → Community Plugins → Browse**
2. Search for "Collapse All Folders"
3. Click **Install**, then **Enable**

### Manual Installation

1. Copy `main.js`, `manifest.json`, and `styles.css` into your vault's `.obsidian/plugins/obsidian-collapse-all/` directory.
2. Enable the plugin in Obsidian's Community Plugins settings.

## Settings

| Setting | Default | Description |
|---|---|---|
| Auto-collapse on vault open | Off | Automatically collapse all folders when the vault is opened. |
